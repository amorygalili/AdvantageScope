# Example React Plugin for AdvantageScope

This is an example React-based plugin for AdvantageScope that demonstrates how to create a custom tab type using React components.

## Building

```bash
# Install dependencies
npm install

# Build the plugin
npm run build

# Watch mode for development
npm run watch
```

The built plugin will be in the `dist` directory.

## Using the Plugin

After building, the plugin can be loaded in AdvantageScope from the _Manage Plugins_ dialog.
